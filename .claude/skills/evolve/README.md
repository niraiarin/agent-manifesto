# evolve

マニフェストに沿って構造を漸進的に改善するスキル。

一時的なインスタンスの連鎖を通じて、永続する構造（スキル、ルール、テスト、フック、文書）の
品質を向上させる。P3 の学習ライフサイクル（観察→仮説化→検証→統合→退役）を
Agent Teams で実行する。

## 起動トリガー

`/evolve` または「改善」「漸進」「自己改善」「構造改善」「進化」

## ワークフロー

```
Observer (観察) → Hypothesizer (仮説化) → Verifier (検証) → Integrator (統合)
```

4つの専門エージェントが順次実行し、各フェーズの成果物を次フェーズに引き渡す。

## 原則

- 改善は漸進的（conservative extension 優先）
- P3 のライフサイクルに従う統治されたプロセス
- このスキル自体も漸進的改善の対象 (D9 自己適用)

## 実績

89回以上の実行実績あり（2026-04-02 時点）。
