# 数理論理学用語リファレンス

本文書は、形式体系の構築に必要な数理論理学および隣接分野の用語を定義する。
各用語の出自（数理論理学・型理論・証明論・モデル理論・科学哲学）を明示する。

---

## 1. 形式体系 (Formal System)

形式体系は以下の 3 要素から構成される。

| 構成要素 | 定義 |
|----------|------|
| **形式言語 (Formal Language)** | 記号の集合と、それらの組み合わせ規則（文法）。どのような式が「整式 (well-formed formula, wff)」であるかを定める |
| **公理 (Axiom)** | 証明なしに真と仮定する整式の集合 |
| **推論規則 (Inference Rule)** | 整式から整式を導く変換規則。前提 (premise) から結論 (conclusion) を得る |

形式体系の中で、公理に推論規則を繰り返し適用して得られる整式を**定理 (theorem)** と呼ぶ。
公理から定理に至る推論規則の適用列を**証明 (proof)** と呼ぶ。

## 2. 命題論理 (Propositional Logic)

### 2.1 基本概念

| 用語 | 定義 |
|------|------|
| **命題 (Proposition)** | 真偽が確定する文。真理値を持つ |
| **命題変数 (Propositional Variable)** | 命題を表す変数。p, q, r 等 |
| **論理結合子 (Logical Connective)** | 命題を組み合わせる演算子。¬（否定）、∧（連言）、∨（選言）、→（含意）、↔（同値） |
| **真理値割り当て (Valuation)** | 各命題変数に真 (T) または偽 (F) を割り当てる関数 |

### 2.2 命題の分類

| 用語 | 定義 |
|------|------|
| **恒真式 (Tautology)** | すべての真理値割り当てのもとで真となる式。例: P ∨ ¬P。命題論理に固有の概念 |
| **矛盾式 (Contradiction)** | すべての真理値割り当てのもとで偽となる式。例: P ∧ ¬P |
| **充足可能式 (Satisfiable Formula)** | 真にする真理値割り当てが少なくとも一つ存在する式 |
| **偶然式 (Contingency)** | 恒真でも矛盾でもない式。真理値割り当てに依存して真偽が変わる |

### 2.3 正規形 (Normal Form)

| 用語 | 定義 |
|------|------|
| **リテラル (Literal)** | 命題変数またはその否定。p, ¬p |
| **連言標準形 (Conjunctive Normal Form, CNF)** | 節（リテラルの選言）の連言として表された式。(p ∨ ¬q) ∧ (r ∨ s) |
| **選言標準形 (Disjunctive Normal Form, DNF)** | 項（リテラルの連言）の選言として表された式。(p ∧ ¬q) ∨ (r ∧ s) |

任意の命題論理式は、論理的に同値な CNF および DNF に変換可能である。

### 2.4 命題間の関係

| 用語 | 定義 |
|------|------|
| **論理的帰結 (Logical Consequence)** | Γ ⊨ φ: Γ を真にするすべての真理値割り当てのもとで φ も真。意味論的概念 |
| **導出可能性 (Derivability)** | Γ ⊢ φ: 前提集合 Γ から推論規則の有限回適用で φ が導出可能。構文論的概念 |
| **意味論的同値 (Semantic Equivalence)** | φ ⊨ ψ かつ ψ ⊨ φ が成立。すべての真理値割り当てで同じ真理値を取る |
| **相互導出可能性 (Interderivability)** | φ ⊢ ψ かつ ψ ⊢ φ が成立。構文論的な同値関係 |

**⊨ と ⊢ の区別:** ⊨ は意味論的（真理値割り当て / 解釈に基づく）、⊢ は構文論的（推論規則に基づく）。
健全性と完全性が成立する体系では、Γ ⊨ φ と Γ ⊢ φ は一致する。

### 2.5 前提集合と条件付き推論

| 用語 | 定義 |
|------|------|
| **前提集合 (Premise Set)** | 導出の出発点として真と仮定する式の集合 Γ。Γ ⊢ φ は「Γ を前提として φ が導出可能」を意味する。Γ が空のとき ⊢ φ と書き、φ は体系の定理 |
| **条件付き導出 (Conditional Derivation)** | Γ ⊢ φ。前提集合 Γ のもとでの導出。φ の真偽は Γ の仮定に依存する |
| **含意 (Implication)** | P → Q。対象言語レベルの条件文。「P ならば Q」。含意は論理結合子（§2.1）の一つであり、導出可能性（⊢）とは異なるレベルの概念 |
| **単調性 (Monotonicity)** | Γ ⊢ φ ならば Γ ∪ {ψ} ⊢ φ。前提の追加は既存の導出を無効にしない。古典論理および直観主義論理の基本性質 |

**含意と条件付き導出の関係:**
対象言語の含意（P → Q）とメタレベルの条件付き導出（Γ, P ⊢ Q）は
異なるレベルの概念だが、演繹定理（§5.3）により対応づけられる:

```
Γ, P ⊢ Q    （メタレベル: P を前提集合に追加して Q を導出）
    ↕         演繹定理（多くの体系で成立）
Γ ⊢ P → Q   （対象言語レベル: 含意 P → Q を導出）
```

**単調性の帰結:**
前提集合の拡大は、既に導出可能な式の導出可能性を保存する。
これは体系の拡大（§5.5）において、元の理論の定理が拡大後も定理であり続ける
根拠となる。

**非単調論理との対比:** デフォルト論理やオートエピステミック論理は単調性を持たず、
前提の追加により既存の結論が撤回されうる。これらは本文書の範囲外である。

## 3. 述語論理 (Predicate Logic)

命題論理を拡張し、対象の内部構造と量化を扱う。一階述語論理 (first-order logic) が標準。

### 3.1 言語の構成要素

| 用語 | 定義 |
|------|------|
| **個体変数 (Individual Variable)** | 論議領域の要素を表す変数。x, y, z |
| **個体定数 (Individual Constant)** | 論議領域の特定の要素を指す記号。a, b, c |
| **関数記号 (Function Symbol)** | 個体から個体への写像を表す記号。f(x), g(x, y)。アリティ（引数の数）を持つ |
| **述語記号 (Predicate Symbol)** | 個体の組に対して真偽を定める記号。P(x), Q(x, y)。アリティを持つ |
| **量化子 (Quantifier)** | ∀（全称量化子: すべての個体について）、∃（存在量化子: ある個体が存在して） |
| **束縛変数 (Bound Variable)** | 量化子の作用域内にある変数。∀x.P(x) における x |
| **自由変数 (Free Variable)** | 量化子に束縛されていない変数。P(x) における x |

### 3.2 意味論（モデル理論の基礎）

| 用語 | 定義 |
|------|------|
| **論議領域 (Domain of Discourse)** | 個体変数が取りうる値の非空集合。構造の台集合 (underlying set) |
| **構造 (Structure)** | 論議領域と、その上での関数記号・述語記号の解釈の組 𝔄 = (A, ...) |
| **解釈 (Interpretation)** | 言語の各記号に意味を割り当てること。構造と変数への付値で構成される |
| **モデル (Model)** | 論理式の集合 Σ を充足する構造。𝔄 ⊨ Σ |
| **妥当 (Valid)** | すべての構造のもとで真。⊨ φ と書く。命題論理の恒真式の一般化 |
| **充足可能 (Satisfiable)** | 真にする構造が少なくとも一つ存在する |

### 3.3 正規形（一階述語論理）

| 用語 | 定義 |
|------|------|
| **冠頭標準形 (Prenex Normal Form)** | すべての量化子が式の先頭に集められた形。Q₁x₁ Q₂x₂ ... Qₙxₙ. φ（φ は量化子を含まない）。任意の一階論理式は論理的に同値な冠頭標準形に変換可能 |
| **スコーレム化 (Skolemization)** | 冠頭標準形の存在量化子を、先行する全称量化変数に依存するスコーレム関数で置き換える操作。充足可能性を保存する（妥当性は保存しない） |

### 3.4 述語・関係・関数

| 用語 | 定義 |
|------|------|
| **述語 (Predicate)** | n 個の個体を受け取り真偽を返す。n 項述語 P(x₁, ..., xₙ)。n をアリティと呼ぶ |
| **単項述語 (Unary Predicate)** | アリティ 1 の述語。個体の性質を表す。P(x) |
| **二項関係 (Binary Relation)** | アリティ 2 の述語。二つの個体の間の関係を表す。R(x, y) |
| **関数 (Function)** | 入力に対して出力が一意に定まる写像。述語論理では関数記号で表される |

## 4. 公理と定理

### 4.1 公理に関する用語

| 用語 | 定義 |
|------|------|
| **公理 (Axiom)** | 形式体系において証明なしに真と仮定する命題 |
| **論理的公理 (Logical Axiom)** | 論理体系自体に属する公理。すべての理論で共有される。例: 命題論理の公理図式 |
| **非論理的公理 (Non-logical Axiom)** | 特定の理論に固有の公理。その理論が扱う対象についての仮定 |
| **公理図式 (Axiom Schema)** | 無限個の公理を一つのパターンで表現したもの。式変数を具体的な式に置き換えて個々の公理を得る |

**歴史的注記:** 「公理 (axiom)」と「公準 (postulate)」はユークリッド『原論』では区別されていた
（共通概念 common notion vs 要請 postulate）。現代の数理論理学では両者は同義であり、
形式的な区別は存在しない。

### 4.2 定理に関する用語

| 用語 | 定義 |
|------|------|
| **定理 (Theorem)** | 公理と推論規則から証明された命題 |
| **補題 (Lemma)** | 定理の証明に用いる中間的な結果。形式的には定理と同等であり、区別は慣習的 |
| **系 (Corollary)** | ある定理から直ちに（あるいは短い追加推論で）導かれる結果 |
| **命題 (Proposition)** | 定理ほど重要でないが独立して述べる価値のある結果。形式的には定理と同等 |

**注記:** 定理・補題・系・命題の区別は純粋に慣習的なものであり、
形式的にはすべて「公理から導出された命題」である。

### 4.3 公理の独立性

| 用語 | 定義 |
|------|------|
| **独立性 (Independence)** | ある公理が、残りの公理から導出できないこと。独立でない公理は冗長である |
| **独立性の証明方法** | 「その公理を否定しても残りの公理と無矛盾であること」をモデルの構成で示す |

**例:** ユークリッドの第 5 公準（平行線公準）は他の 4 公準から独立している。
否定しても無矛盾な体系（非ユークリッド幾何学）が存在することで証明された。

## 5. 形式体系のメタ性質

形式体系自体の性質を論じるのが**メタ理論 (metatheory)** である。

### 5.1 基本的メタ性質

| 用語 | 定義 |
|------|------|
| **無矛盾性 (Consistency)** | 体系から矛盾（φ ∧ ¬φ）が導出されないこと。爆発律 (ex falso quodlibet) を持つ体系（古典論理・直観主義論理）では、「ある式 φ について ⊬ φ」と同値 |
| **健全性 (Soundness)** | 証明可能なものはすべて妥当。Γ ⊢ φ ならば Γ ⊨ φ |
| **完全性 (Completeness)** | 妥当なものはすべて証明可能。Γ ⊨ φ ならば Γ ⊢ φ |
| **決定可能性 (Decidability)** | 任意の式について、定理であるか否かを判定するアルゴリズムが存在する |

### 5.2 一階述語論理の基本定理

| 定理 | 内容 |
|------|------|
| **ゲーデルの完全性定理 (Gödel, 1930)** | 一階述語論理は完全である。すなわち、妥当な式はすべて証明可能。⊨ φ ならば ⊢ φ |
| **コンパクト性定理 (Compactness Theorem)** | 論理式の集合 Σ がモデルを持つための必要十分条件は、Σ のすべての有限部分集合がモデルを持つことである |
| **レーヴェンハイム＝スコーレム定理（下方）(Downward Löwenheim–Skolem)** | 可算言語の一階理論が無限モデルを持つならば、可算モデルを持つ |
| **レーヴェンハイム＝スコーレム定理（上方）(Upward Löwenheim–Skolem)** | 言語 L の一階理論が濃度 κ の無限モデルを持つならば、任意の濃度 λ ≥ max(\|L\|, κ) のモデルを持つ |

**命題論理の決定可能性:** 命題論理は決定可能である（真理値表による判定）。
一階述語論理は半決定可能である（妥当な式を列挙できるが、妥当でない式の判定は保証されない）。

### 5.3 基本的メタ定理

| 定理 | 内容 |
|------|------|
| **演繹定理 (Deduction Theorem)** | Γ, φ ⊢ ψ ならば Γ ⊢ φ → ψ。仮定を前件に移動できる。逆（Γ ⊢ φ → ψ ならば Γ, φ ⊢ ψ）はモーダスポネンスにより常に成立する。多くのヒルベルト流体系で成立するが、すべての形式体系で成立するわけではない |
| **単調性 (Monotonicity)** | Γ ⊢ φ ならば任意の ψ について Γ ∪ {ψ} ⊢ φ。前提の追加は既存の導出を保存する。古典論理および直観主義論理で成立する。体系の拡大（§5.5）が既存の定理を保存する根拠 |

### 5.4 不完全性定理

**ゲーデルの第一不完全性定理 (Gödel, 1931):**
再帰的に公理化可能で、ロビンソン算術 Q を解釈できる無矛盾な形式体系には、
その体系内で証明も反証もできない命題が存在する。

**ゲーデルの第二不完全性定理:**
上記の条件を満たす無矛盾な形式体系は、自身の無矛盾性を体系内で証明できない。

**適用条件の注記:** 不完全性定理は十分に強い体系（少なくともロビンソン算術 Q を
解釈できる体系）にのみ適用される。命題論理や、十分に弱い理論はこの定理の対象外であり、
完全かつ決定可能でありうる。

### 5.5 体系の拡張

| 用語 | 定義 |
|------|------|
| **拡大 (Extension)** | 体系 S₂ が体系 S₁ の拡大であるとは、S₁ のすべての定理が S₂ でも定理であること。Thm(S₁) ⊆ Thm(S₂) |
| **保存拡大 (Conservative Extension)** | S₁ の言語で表現可能な式について、S₂ で新たに証明可能になるものがない拡大。すなわち、S₁ の言語 L₁ の式 φ について、S₂ ⊢ φ ならば S₁ ⊢ φ |
| **無矛盾な拡大 (Consistent Extension)** | 拡大後の体系 S₂ が無矛盾である拡大。S₁ が無矛盾であることを前提とする |
| **定義的拡大 (Definitional Extension)** | 新しい記号を既存の記号で定義することによる拡大。常に保存拡大 |

**関係:** S₁ が無矛盾であるとき、定義的拡大 ⊂ 保存拡大 ⊂ 無矛盾な拡大。
保存拡大は無矛盾な拡大であるが、逆は成立しない。
無矛盾な拡大は元の言語について新しい定理を追加しうるが、保存拡大は追加しない。

### 5.6 対象理論とメタ理論

| 用語 | 定義 |
|------|------|
| **対象理論 (Object Theory)** | 研究対象となる形式体系そのもの |
| **メタ理論 (Metatheory)** | 対象理論の性質を論じる理論。「体系は無矛盾である」はメタ定理 |
| **メタ定理 (Metatheorem)** | メタ理論の中で証明される定理。例: 「一階述語論理は完全である」 |
| **対象言語 (Object Language)** | 対象理論の言語 |
| **メタ言語 (Metalanguage)** | メタ理論を記述する言語 |

## 6. 証明論 (Proof Theory)

### 6.1 証明体系

| 用語 | 定義 |
|------|------|
| **ヒルベルト流体系 (Hilbert-style System)** | 多数の公理図式と少数の推論規則（主にモーダスポネンス）から構成される証明体系 |
| **自然演繹 (Natural Deduction)** | 各論理結合子に対する導入規則と除去規則の対を基本操作とする証明体系。ゲンツェン (Gentzen, 1935) |
| **シーケント計算 (Sequent Calculus)** | シーケント Γ ⊢ Δ（前件と後件の式の列）を基本単位とする証明体系。ゲンツェン (Gentzen, 1935)。LK（古典論理）、LJ（直観主義論理）が代表的 |

### 6.2 証明の性質

| 用語 | 定義 |
|------|------|
| **仮定 (Assumption / Hypothesis)** | 証明の中で一時的に真と認める命題 |
| **放出 (Discharge)** | 仮定を前件として含意の中に取り込み、仮定への依存を解消すること |
| **構成的証明 (Constructive Proof)** | 存在を主張する場合に具体的な証拠（witness）を構成する証明。排中律を仮定しない |
| **非構成的証明 (Non-constructive Proof)** | 背理法などにより存在を示すが、具体的な証拠を与えない証明 |

### 6.3 カット除去

| 用語 | 定義 |
|------|------|
| **カット規則 (Cut Rule)** | シーケント計算において、中間的な式（カット式）を介して二つの証明を結合する規則。Γ ⊢ A, Δ と Γ', A ⊢ Δ' から Γ, Γ' ⊢ Δ, Δ' を導く |
| **カット除去定理 (Cut Elimination, Hauptsatz)** | カット規則を含む証明は、カット規則を使わない証明に変換可能。ゲンツェン (Gentzen, 1935)。証明の正規化に相当する |
| **部分式性 (Subformula Property)** | カットなし証明では、証明中に現れるすべての式が結論の部分式である。カット除去の重要な帰結 |

### 6.4 空虚な推論

| 用語 | 定義 |
|------|------|
| **空虚な真 (Vacuous Truth)** | 含意 P → Q において前件 P が偽であるとき、含意全体は真。Q の内容に関係なく成立する |
| **非空虚性 (Non-vacuity)** | 公理が自明に真ではないこと。前件が充足可能であること |

## 7. 型理論 (Type Theory)

型理論は集合論とは異なる数学の基礎づけを提供する。

**集合論との違い:** 集合論ではすべてが集合であり、所属関係 (∈) が基本。
型理論では「項 (term)」と「型 (type)」の判定関係 (: — 「は〜型を持つ」) が基本であり、
すべての項は生まれながらに型を持つ。

### 7.1 基本概念

| 用語 | 定義 |
|------|------|
| **型 (Type)** | 項の分類。項は正確に一つの型に属する |
| **項 (Term)** | 型の要素。計算の対象 |
| **型判定 (Typing Judgment)** | t : T — 項 t が型 T を持つことの宣言 |
| **関数型 (Function Type)** | A → B — 型 A の項を受け取り型 B の項を返す関数の型 |
| **依存型 (Dependent Type)** | 戻り値の型が引数の値に依存する関数型。Π (x : A), B(x) |
| **依存和型 (Dependent Sum Type)** | 第二成分の型が第一成分の値に依存する対の型。Σ (x : A), B(x) |
| **命題型対応 (Propositions as Types)** | カリー＝ハワード対応 (Curry-Howard Correspondence)。命題を型、証明を項として同一視する。含意は関数型、全称量化は依存型に対応 |

### 7.2 帰納的構成 (Inductive Construction)

| 用語 | 定義 |
|------|------|
| **帰納型 (Inductive Type)** | 構成子 (constructor) の有限集合で定義される型。各構成子は型の要素を生成する規則 |
| **構成子 (Constructor)** | 帰納型の要素を生成する関数 |
| **再帰子 (Recursor / Eliminator)** | 帰納型に対するパターンマッチと再帰を行う消去原理。帰納型の定義から生成される |
| **構造体 (Structure / Record Type)** | 単一の構成子を持つ帰納型。名前付きフィールドの直積 |

### 7.3 CIC 固有の概念

Calculus of Inductive Constructions (CIC) は、Coq や Lean の基盤となる型理論。

| 用語 | 定義 |
|------|------|
| **宇宙 (Universe)** | 型の型。Type₀ : Type₁ : Type₂ : ... の階層を成す。宇宙多相 (universe polymorphism) により柔軟に扱う |
| **Prop** | 命題の宇宙。Lean 4 では証明無関係 (proof irrelevance) が定義的に成立する。Coq では公理として追加可能だが組み込みではない |
| **帰納的族 (Inductive Family)** | パラメータやインデックスに依存する帰納型の族 |

## 8. 計算可能性との接続

数理論理学と計算可能性理論は密接に関連する。

| 用語 | 定義 |
|------|------|
| **チャーチ＝チューリングのテーゼ (Church-Turing Thesis)** | 直観的に「計算可能」な関数はすべてチューリング機械で計算可能。テーゼであり定理ではない |
| **決定問題 (Decision Problem)** | 任意の入力に対して Yes/No を返すアルゴリズムが存在するかという問題 |
| **停止問題 (Halting Problem)** | チューリング機械が与えられた入力に対して停止するかの決定問題。決定不能 (Turing, 1936) |
| **再帰的集合 (Recursive/Decidable Set)** | 所属判定が計算可能な集合。体系の決定可能性と直結する |
| **再帰的枚挙可能集合 (Recursively Enumerable Set)** | 要素を列挙するアルゴリズムが存在する集合。半決定可能に対応 |

**論理学との関係:** 一階述語論理の妥当な式の集合は再帰的枚挙可能だが再帰的でない
（半決定可能だが決定不能）。ゲーデルの不完全性定理の証明は原始再帰関数を用いる。

## 9. 隣接分野の用語

以下の用語は厳密には数理論理学に属さないが、形式体系の設計・運用で頻繁に参照される。
出自を明記して収録する。

### 9.1 科学哲学 (Philosophy of Science)

| 用語 | 定義 | 出自 |
|------|------|------|
| **反証可能性 (Falsifiability)** | 命題を偽とする観察が原理的に可能であること。科学的命題の画定基準 | ポパー (Popper, 1934) |
| **経験的命題 (Empirical Proposition)** | 観察・実験により真偽が検証される命題 | 経験主義全般 |
| **分析的命題 (Analytic Proposition)** | 意味のみから真偽が定まる命題。形式体系の公理はこれに近い | カント以降 |

**数理論理学との関係:** 形式体系内の公理はすべて体系内で真と仮定されるため、
形式的には反証可能性の概念が適用されない。しかし、公理を現実世界の主張として
解釈する場合、その主張の反証可能性を問うことは科学哲学的に意味がある。

### 9.2 信念修正理論 (Belief Revision Theory)

| 用語 | 定義 | 出自 |
|------|------|------|
| **拡張 (Expansion)** | 信念集合に新しい信念を追加 | AGM (1985) |
| **縮小 (Contraction)** | 信念集合から信念を除去 | AGM (1985) |
| **修正 (Revision)** | 信念集合に新しい信念を追加し、無矛盾性を維持するよう既存の信念を調整 | AGM (1985) |

**数理論理学との関係:** AGM 理論は命題論理の上に構築されるが、
標準的な数理論理学の一部ではなく、形式認識論 (formal epistemology) に属する。
体系の公理を「信念」と見なす場合に、公理の追加・除去・修正の操作を定式化できる。

### 9.3 プログラム検証 (Program Verification)

| 用語 | 定義 | 出自 |
|------|------|------|
| **不変条件 (Invariant)** | プログラムの実行中（あるいはループの各反復で）常に保持される性質 | フロイド＝ホーア (Floyd-Hoare) |
| **事前条件 (Precondition)** | 操作の実行前に成立すべき条件 | ホーア論理 |
| **事後条件 (Postcondition)** | 操作の実行後に成立する条件 | ホーア論理 |
| **遷移関係 (Transition Relation)** | 状態 s から状態 s' への遷移を表す関係 R(s, s') | 状態遷移系 |

### 9.4 証明支援系 (Proof Assistants)

以下の概念は数学的に基礎を持つが、証明支援系（Coq, Lean, Agda 等）の実装に固有の用語として使われる。

| 用語 | 定義 | 出自 |
|------|------|------|
| **型クラス (Typeclass)** | 型に対するインタフェース。型が満たすべき演算や性質の仕様を定め、インスタンス解決により自動推論される | Haskell / Lean / Coq |
| **不透明定義 (Opaque Definition)** | 型のみが公開され定義本体が隠蔽される定義。体系は存在と型のみを知る | 証明支援系全般 |
| **タクティク (Tactic)** | 証明を段階的に構成するためのメタプログラム。証明状態を変換する | LCF 系 / Coq / Lean |

### 9.5 測度論・最適化理論

| 用語 | 定義 | 出自 |
|------|------|------|
| **可測関数 (Measurable Function)** | 測度空間の間の、可測集合の逆像が可測集合である関数 | 測度論 |
| **目的関数 (Objective Function)** | 最適化問題において最大化または最小化する関数 | 最適化理論 |

**注意:** 日常的に「測定可能な関数」の意味で「可測関数」を使う場合があるが、
測度論の可測関数とは厳密に異なる概念である。

## 10. 用語間の関係図

### 10.1 形式体系の内部構造

```
┌─────────────────────────────────────────────────────┐
│                 形式体系 (Formal System)              │
│                                                      │
│  形式言語 ──定義──→ 整式 (Well-formed Formula)       │
│                          │                           │
│                    ┌─────┴─────┐                     │
│                    │           │                     │
│                 公理          推論規則                │
│              (仮定する)     (導出する)                │
│                    │           │                     │
│                    └─────┬─────┘                     │
│                          │                           │
│                          ▼                           │
│                        定理                          │
│                     (導出された命題)                   │
│                          │                           │
│                 ┌────────┼────────┐                   │
│                 │        │        │                   │
│               補題      系     命題                   │
│             (中間結果) (直接帰結) (独立結果)            │
└─────────────────────────────────────────────────────┘
```

### 10.2 構文論と意味論の対応

```
    構文論 (Syntax)              意味論 (Semantics)
    ──────────────              ─────────────────
    整式 (wff)         ←──→     解釈のもとでの真理値
    論理的公理         ←──→     すべての構造で真（妥当）
    非論理的公理       ←──→     理論のモデルを限定する条件
    導出可能 Γ ⊢ φ    ←──→     論理的帰結 Γ ⊨ φ
    相互導出可能       ←──→     意味論的同値

    これらの対応を保証するメタ定理:
      健全性: Γ ⊢ φ ⇒ Γ ⊨ φ（常に望ましい性質）
      完全性: Γ ⊨ φ ⇒ Γ ⊢ φ（一階述語論理で成立、高階では一般に不成立）
```

### 10.3 論理体系の系譜

以下は歴史的発展と表現力の関係を示す。矢印は「拡張」または「定式化」を表す。

```
    命題論理 (Propositional Logic)
        │ 拡張（個体への量化を追加）
        ▼
    一階述語論理 (First-order Logic)
        │ 拡張（述語・関数への量化を追加）
        ▼
    高階論理 (Higher-order Logic)
        └── 単純型理論 (Simple Type Theory, Church 1940)
              高階論理の代表的定式化

    ─── 別系統 ───

    型付きラムダ計算 (Typed Lambda Calculus)
        │ 拡張（依存型の導入、ラムダキューブ）
        ▼
    依存型理論 (Dependent Type Theory)
        │ 拡張（帰納型の追加）
        ▼
    CIC (Calculus of Inductive Constructions)
```

**注記:** 高階論理と依存型理論は異なる系譜に属する。
高階論理は述語論理の量化の拡張として発展し、
依存型理論は型付きラムダ計算の拡張として発展した。
両者は表現力において重なる部分があるが、基盤となる概念が異なる。

### 10.4 型理論における対応（カリー＝ハワード）

```
    論理               型理論              計算
    ────               ──────              ────
    命題          ←→   型            ←→   仕様
    証明          ←→   項            ←→   プログラム
    含意 A → B    ←→   関数型 A → B  ←→   関数
    連言 A ∧ B    ←→   直積型 A × B  ←→   ペア
    選言 A ∨ B    ←→   直和型 A + B  ←→   タグ付き共用体
    全称 ∀x.P(x)  ←→   依存型 Πx.B(x)←→  多相関数
    存在 ∃x.P(x)  ←→   依存和 Σx.B(x)←→  パッケージ
    仮定の放出     ←→   λ 抽象        ←→   関数定義
    モーダスポネンス←→   関数適用       ←→   関数呼び出し
```
